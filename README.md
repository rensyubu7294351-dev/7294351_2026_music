
## GitHub Pages 公開設定手順

**① リポジトリにファイルをアップロードする**

まだの場合、以下の3ファイルをリポジトリに入れてください。

```
7294351_2026_music/
├── index.html         ← 今回修正したもの
├── Shichifuku_2026.mp3
└── images/
    └── background.jpg
```

GitHubのリポジトリページを開いて「Add file」→「Upload files」でドラッグ＆ドロップできます。

---

**② GitHub Pagesを有効にする**

1. リポジトリページの上部タブ「**Settings**」をクリック
2. 左メニューの「**Pages**」をクリック
3. 「Branch」のところが `None` になっているので、**`main`** に変更
4. 「**Save**」をクリック

---

**③ URLが発行される**

数分待つと以下のURLでアクセスできるようになります。

```
https://rensyubu7294351-dev.github.io/7294351_2026_music/
```
